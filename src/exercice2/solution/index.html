Jeu de memory

Règles du jeu

Toutes les cartes sont étalées faces cachées. Retourne deux cartes. Si c'est la même image qui apparaît sur les deux cartes, tu gagnes les cartes. Si les deux cartes ne vont pas ensemble, elles disparaissent au bout de 2 secondes et tu devras retenter ta chance. Essayes de retrouver les cartes le plus rapidement possible.
